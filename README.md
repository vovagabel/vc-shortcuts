# Custom Shortcodes for Visual Composer

#### Textfield
```
array(
  'type' => 'textfield',
  'heading' => __( 'Text', 'js_composer' ),
  'param_name' => 'text',
  'description' => __( '', 'js_composer' ),
),
```
#### Button
```
array(
  'type' => 'vc_link',
  'heading' => 'Button',
  'param_name' => 'button',
  'description' => __( '', 'js_composer' ),
),
```

#### Add param 
`'admin_label' => true,`
